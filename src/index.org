** Edit Workshop
*** Steps
+ Step 1 :: Go to the URL [[http://outreach.vlabs.ac.in/][http://outreach.vlabs.ac.in/]]

+ Step 2 :: Login as Outreach Coordinator (Please follow the steps mentioned in section [[Login Scenarios]])

+ Step 3 :: Click on =Manage Workshops= button from left menu bar. 

+ Step 4 :: Click on =Pending Workshops= and select the edit
            button of the respective workshop to edit the
            details of a Workshop.

+ Step 5 :: Update any of the following details of the Workshop and Click on =Submit=
     button.
            + Workshop Name - <workshop name>
            + Location of Workshop - <location of workshop>
            + Date of Workshop - <date of workshop>
            + Participating Institutes - <participating institutes>
            + Number of Participants Attended - <number of expected participants>
            + Number of Sessions - <number of sessions>
            + Duration of Each Session - <duration of each session>
            + Number of Labs conducted in the workshop - <number of labs conducted>
            
	    + For online workshop
              - Gateway IP of College - Click on =Get Gateway IP= button
                to get the gateway ip of college where the workshop is
                conducted.  *Note* This feature is useful only if you are in the site
                where the workshop is being conducted.
	      - Copy & paste the obtained gateway ip address into the input field.
              - Usage from Online Feedback 
              - Click on =Compute Usage= button to get the Usage from
                online feedback which is auto populated based on given
                =workshop date= and =college gateway ip=
              - Usage from Paper Feedback - <usage from paper feedback> *Note* Update this field if the online
                usage is not correct.
            + For offline workshop
	      - Enter the =College MAC Address= value which is
                obtained when the workshop was conducted at the college. Visit [[http://community.vlabs.ac.in/docs/user-manuals/college-cloud/installation-manual.html#sec-7-2][More information on getting MAC address]]
              - Click on =Compute Usage= button to get the Usage from
                college cloud feedback which is auto populated based on given
                =workshop date= and =mac address=.

            + Usage from Paper Feedback - <usage recorded>

+ Step 6 :: Updated Workshop will appear under =Upcoming Workshops=
            section if the workshop date is future. or else if the
            workshop is already done, it will appear under =Pending
            Action Workshops= section.

+ Step 7 :: Click on =Logout= button from top right corner to exit. or
            else continue with other scenarios

** Approve/Disapprove the Nodal Coordinator Workshop
*** Steps
+ Step 1 :: Go to the URL [[http://outreach.vlabs.ac.in/][http://outreach.vlabs.ac.in/]]

+ Step 2 :: LogIn as Outreach Coordinator (Please follow the steps mentioned in [[Login Scenarios]])

+ Step 3 :: Click on =Manage Workshops= button from left menu bar.

+ Step 4 :: Outreach Coordinator =Manage Workshops= will show the following Details:
            1) Upcoming Workshops 
            2) Pending Workshops
            3) Nodal Coordinators Workshops
            4) Workshop History
            5) ADD WORKSHOP

+ Step 5 :: Click on =Nodal Coordinators Workshops= will show
            the following Details:
            + Coordinator Name - <Coordinator Name>
            + Workshop Name - <Name of the workshop>
            + Location - <Location of the workshop>
            + Phone Number - <Phone number of the Nodal
              Coordinator>
            + User status - < Active/Inactive status of the
              Nodal Coordinator>
            + Last Active - <Time, Date and day in the week>
            + Actions - <Edit options to update the details
              of Nodal Coordinator>

+ Step 6 :: Search based on =date= and click on =View=
            button, verify the details and documents related
            to the workshop. 

+ Step 7 :: If details and documents are uploaded correctly
            click on =Approve= button. Otherwise describe the
            missing details/ documents and disapprove the
            workshop by clicking =Disapprove= button

+ Step 8 :: Click on =Logout= button from top right corner to exit. or else
     continue with other scenarios

** Manage Workshops
*** Steps
+ Step 1 :: Go to the URL [[http://outreach.vlabs.ac.in/][http://outreach.vlabs.ac.in/]]

            steps mentioned in [[Login Scenarios]])

            to search the Workshops as mentioned below
            + Upcoming Workshops

            + Workshop History

            + Coordinator Name - <Coordinator Name>

            and =Sort Descending=. It also allows to hide

            + Location - <Location of the workshop>
            + Participants - <No of participants in the workshop>
            + Date - <Date of the Workshop>
            + Workshop status - < Active/Inactive status of the
              workshop>
            + Status - <Status of the workshop>
            + Usage - <Usage obtained in the workshop>

+ Step 7 :: To Export the data, click on the menu button on
            the right corner of the table. The following
            are features:
            + Clear all filters
            + Export all data as csv
            + Export the visible data as csv
            + Hide the columns and export the required
              columns

+ Step 8 :: Click on =Logout= button from top right corner to exit. or else
     continue with other scenarios

** Add Workshop 
*** Steps
+ Step 1 :: Go to the URL [[http://outreach.vlabs.ac.in/][http://outreach.vlabs.ac.in/]]

+ Step 2 :: Login as Nodal Coordinator (Please follow the steps mentioned in [[Login Scenarios]])

+ Step 3 :: Click on =Manage Workshops= button from left menu bar. 

+ Step 4 :: Click on =Add Workshop= button.

+ Step 5 :: Enter the following details to create a New Workshop and
            Click on =Submit= button. 

            + Workshop Name - <workshop name>
            + Location of Workshop - <location of workshop>
            + Workshop Institute Name - <participating institutes>
            + Number of Expected Participants - <number of expected participants>
            + Number of Sessions - <number of sessions>
            + Duration of Each Session - <duration of each session>
            + Number of Labs Planned - <number of labs planned>
            + Date of Workshop - <date of workshop>

+ Step 6 :: Newly added Workshop will appear under =Upcoming
            Workshops= section if the workshop date is future. or else
            if the workshop is already done, it will appear under
            =Pending Workshops= section.

+ Step 7 :: Click on =Logout= button from top right corner to exit. or
            else continue with other scenarios
